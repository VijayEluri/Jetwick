  • Jetwick is an open source twitter search and offers the following features:
    • Find relevant tweets fast! Even old ones.
    • Filter tweets by date, language, keywords, ...
    • Search your own or others tweets.
    • For the git repository look at this page
    • Updates are tweeted @jetwick and blogged from karussell
    • For more features with a video and screenshots look at this page from pannous.info
  • Advertisment. It is easy to create an advertisment for jetwick.com:
    • Create a tweet with your headline and add a link if you like. Then add the hashtag #jetwick and your own hashtags (which represents the keywords where you want that your ad will be shown too)
    • If this tweet is then retweeted it is likely to be shown. (At the moment only one retweet is necessary. This will be increased in the future.) Your tweet then will 'fight' against the other advertisments booked for the entered keyword. The tweet with the most retweets will win. (This will probably change in future so that more relevant tweets will win) If two or more tweets have the same retweet number both will have the same probability to be shown.
    • After 3 days the advertisment is removed.
    • This principle was first invented by stackoverflow.
  • Why we named it 'Jetwick'?
    You might think that we combined jetty and wicket, some Java software we use. But the truth is that the initial name was 'Yatwic' which we derived from 'Yet Another Twitter Client'. The spelling was a bit awkward so we changed it to its faster brother Jetwick.
    By the way: At the beginning we didn't use wicket.
