  • Jetwick is an open source twitter search and offers the following features:
    • Find relevant tweets fast! Even old ones.
    • Filter tweets by date, language, keywords, ...
    • Search your own or others tweets.
    • For the git repository look at this page
    • Updates are tweeted @jetwick and blogged from karussell
    • For more features with a video and screenshots look at this page from pannous.info
  • Why we named it 'Jetwick'?
    You might think that we combined jetty and wicket, some Java software we use. But the truth is that the initial name was 'Yatwic' which we derived from 'Yet Another Twitter Client'. The spelling was a bit awkward so we changed it to its faster brother Jetwick.
    By the way: At the beginning we didn't use wicket.
